# injector

capacitor plugin to inject js on app start up

## Install

```bash
npm install injector
npx cap sync
```

## API
In `capacitor.config.json`:

```json
{
  "plugins": {
    "Injector": {
      "ios": 'some js code for ios',
      "android": 'some js code for android'
    }
  }
}
```
